# Full-Stack Project

A web application that displays list of restaurants.

## Requirements

- Docker
- Docker-compose

## Start app

At root run the following command to start the app in the development mode.

`Docker-compose up`

Access frontend in
http://localhost:3000

Access backend in
http://localhost:8000
